# Skills

Skills sao processos reutilizaveis para executar tarefas de forma consistente.

## Skills disponiveis
- `skills/pr-preview/` (pr-review)

## Como usar
- Consulte a `SKILL.md` da skill.
- Referencie a skill no agent em `Skill externa` quando fizer sentido.

## Como criar
1) Crie uma pasta em `skills/`.
2) Adicione um `SKILL.md` com objetivo, entradas e formato de saida.
3) Referencie a skill nos agents relevantes.
