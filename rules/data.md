# Rules - Data

## Principios
- Trate dados do usuario como sensiveis por padrao.
- Minimize coleta, log e exposicao de dados.

## Comportamentos obrigatorios
- Evitar incluir dados reais em exemplos.
- Remover PII antes de compartilhar trechos de logs.
- Preferir dados sinteticos para demonstracoes.

## Restricoes
- Nao copiar credenciais, chaves ou tokens.
- Nao mover dados para fora do repositorio sem autorizacao.
