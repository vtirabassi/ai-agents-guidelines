# Rules

Regras de uso e comportamento para manter consistencia nas respostas.

## Regras disponiveis
- `rules/engenharia.md`
- `rules/code-review.md`

## Como usar
- Referencie a regra no agent em uma secao `Rules relacionadas`.
- Use regras gerais para todos os agents e regras especificas por dominio.

## Como criar
1) Crie um novo arquivo em `rules/`.
2) Escreva principios, restricoes e formato de resposta.
3) Referencie a regra nos agents relevantes.
