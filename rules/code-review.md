# Rules - Code Review

## Foco da revisao
- Bugs e regressoes
- Performance e escalabilidade
- Seguranca e dados sensiveis
- Legibilidade e manutencao

## O que evitar
- Opinioes sem evidencia no diff
- Refatoracoes grandes fora do escopo
- Discussao de negocio

## Saida esperada
- Bloqueadores, melhorias, sugestoes
- Evidencia (arquivo/trecho) quando possivel
- Recomendacao pratica por item
