# Getting Started

Guia rapido para adotar o repositorio com o time em 10 minutos.

## 1) Ler o minimo necessario (2 min)
- `README.md`
- `docs/skills.md`
- `docs/agents.md`
- `docs/rules.md`

## 2) Escolher o modo de uso (2 min)
- Se a tarefa e recorrente: usar uma skill.
- Se precisa de comportamento especializado: usar um agent.
- Em todos os casos: respeitar `rules/`.

## 3) Executar um primeiro caso real (3 min)
- Exemplo A: review de PR com `skills/pr-review/SKILL.md`.
- Exemplo B: status de incidente com `skills/incident-status-report/SKILL.md`.
- Exemplo C: resumo de conteudo com `skills/content-summary/SKILL.md`.

## 4) Validar qualidade da resposta (2 min)
Checklist minimo:
- A saida seguiu o formato esperado?
- Nao inventou contexto/dados?
- Ficou acionavel para o proximo passo?

## 5) Evoluir por iteracao curta (1 min)
- Abrir PR pequeno com ajuste objetivo.
- Atualizar docs relacionados se houver mudanca de contrato.

## Padrao de idioma
- Nome de arquivos, pastas, skills e agents: ingles.
- Conteudo textual: PT-BR por padrao do time, salvo decisao diferente.
