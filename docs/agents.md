# Agents

## O que sao?
Agents sao configuracoes reutilizaveis que padronizam o comportamento da IA para o time.

Agente = Skill + Contexto + Objetivo

Eles combinam:
- Skills: como executar a tarefa
- Contexto: codigo, dados, historico
- Objetivo: o que precisa ser entregue

## Na pratica
Agentes evitam que cada pessoa use IA de um jeito diferente.

- Menos improviso
- Mais previsibilidade
- Qualidade escalavel

> Um agente e como um servico com contrato claro, nao um script solto.
