# Rules

## O que sao?
Rules sao diretrizes declarativas que definem:
- Como o agente se comporta
- Seus limites
- Suas prioridades
- O estilo de resposta

## Tipos comuns de Rules

- Safety
  Bloqueiam acoes perigosas ou indevidas
- Policy / Compliance
  Exigem consentimento, logs, retencao e auditoria
- UX / Tone
  Idioma, nivel de detalhe e estilo de comunicacao
- Tooling / Operational
  Quais ferramentas o agente pode ou nao usar
- Data
  Restringem acesso e uso de dados sensiveis

## Na pratica
Rules definem os limites operacionais do agente.

> Rules sao para agentes o que politicas de seguranca e guardrails sao para sistemas distribuidos.

As rules podem ser carregadas:
- Sempre
- Ou sob demanda
